| Other           |         |            |             |           |
| TOTAL DEBT      |         |            |             |           |


--- PAYOFF STRATEGIES ---

AVALANCHE METHOD (saves the most money):
1. Pay minimums on all debts
2. Extra money → highest interest rate first
3. When paid, roll to next highest rate

SNOWBALL METHOD (most motivating):
1. Pay minimums on all debts
2. Extra money → smallest balance first
3. When paid, roll to next smallest

6. 50/30/20 BUDGET CALCULATOR
=========================================

A simple framework for your income.

Monthly After-Tax Income: $________


--- YOUR ALLOCATION ---

| Category        | Target % | Target $ | Actual $ | Difference |
|-----------------|----------|----------|----------|------------|
| NEEDS           |    50%   |          |          |            |
| WANTS           |    30%   |          |          |            |
| SAVINGS/DEBT    |    20%   |          |          |            |


--- WHAT COUNTS AS WHAT ---

NEEDS (50%) — Essential, must-pay:
• Housing (rent/mortgage)
• Utilities
• Groceries
• Transportation (to work)
• Required insurance
• Minimum debt payments
• Basic phone/internet

WANTS (30%) — Nice, not essential:
• Dining out
• Entertainment
• Shopping (non-essential)
• Hobbies
• Subscriptions
• Vacations
• Upgrades

SAVINGS/DEBT (20%) — Future you:
• Emergency fund
• Retirement contributions
• Extra debt payments
• Investments
• Big purchase savings

WHY THIS WORKS
=========================================

Tracking isn't about restriction.
It's about awareness.

Most people don't know where their 
money goes. They earn $5,000/month 
and wonder why they have $200 left.

When you write it down, you see it.
When you see it, you can change it.

Start messy. Improve over time.
The best budget is the one you'll 
actually use.
